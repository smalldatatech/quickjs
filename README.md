# quickjs

[https://bellard.org/quickjs/](https://bellard.org/quickjs/)

The main documentation is in doc/quickjs.pdf or doc/quickjs.html.

## compile

### ubuntu 64 bit

```
$ sudo apt-get install gcc-multilib
$ make
```

## run

```
$ ./qjs examples/hello.js
$ ./qjs -m examples/hello_module.js
```
